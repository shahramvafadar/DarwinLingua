# Product Vision

## Product Overview

This product is a cross-platform learning application focused on German language learning for immigrants, newcomers, and other learners who need practical and structured German support in daily life.

The first core of the product is vocabulary learning with multilingual meanings, example sentences, audio playback, CEFR-based categorization, and topic-based organization.

The long-term vision expands beyond vocabulary learning into a broader practical platform that supports learners in real-world situations such as work, government offices, healthcare, transportation, housing, and social integration.

The application will be built for:

- Android
- iOS
- Windows
- Web (later phase)

The product should be designed from the beginning in a way that maximizes code sharing across mobile, desktop, and future web applications.

---

## Core Vision

The product should help users not only learn German as an academic subject, but also use German in real life.

This means the system should eventually combine:

- language learning
- vocabulary building
- contextual example sentences
- pronunciation support
- guided practice
- real-life topic organization
- immigrant support information

---

## Product Goals

### Primary Goals

- Help users learn German vocabulary from A1 to C2
- Support multiple meaning languages for the same German word
- Allow users to select one or two target meaning languages at the same time
- Provide practical example sentences for each word
- Provide pronunciation support for words and sentences
- Support learning by CEFR level
- Support learning by real-world topics

### Secondary Goals

- Add review and practice features
- Track user progress
- Support favorites and personal lists
- Allow content growth over time through structured imports
- Enable future web platform reuse

### Long-Term Goals

- Provide life-support information for immigrants and foreigners
- Help users find useful local resources such as language schools, conversation cafés, counseling centers, and other support services
- Support search by topic and location
- Optionally support a sustainable revenue model such as ads or low-cost premium features

---

## Target Users

### Primary User Groups

- Immigrants living in German-speaking countries
- Refugees and newcomers
- Foreign workers
- International students
- Family members of immigrants
- Self-learners of German

### Secondary User Groups

- Teachers
- Volunteers helping immigrants
- Support organizations
- Users learning German from multilingual backgrounds

---

## Product Differentiators

The product should differentiate itself through the following:

### 1. Multilingual Meaning Support

Users can choose one or two meaning languages simultaneously, such as:

- Persian + English
- Arabic + English
- Turkish + English
- Kurdish + German explanation (future)
- Russian + English

### 2. Real-Life Context Focus

The content should not remain only academic. It should support practical contexts such as:

- doctor
- school
- job interview
- rental housing
- supermarket
- transport
- office communication
- daily social communication

### 3. Dual Navigation Model

Users should be able to explore vocabulary in two major ways:

- by CEFR level
- by topic

### 4. Expandable Platform Design

The system should be designed so that future modules can be added without architectural damage.

Examples:

- practice engine
- local support discovery
- personalized review
- web platform
- admin panel
- AI-assisted content workflows

---

## Product Principles

- Content-first architecture
- Clean and extensible domain design
- Shared business logic across platforms
- Strong localization support
- Offline-friendly design where possible
- Practical learning over decorative complexity
- Incremental delivery over oversized initial scope

---

## Non-Goals for the First Phase

The first phase should not try to solve everything.

The following are not first-phase priorities:

- full grammar course
- social network features
- advanced gamification
- user-generated public content
- live tutoring
- video lessons
- AI chat tutor
- map-heavy immigrant support system
- complex monetization
