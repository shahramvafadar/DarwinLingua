# Early Product Decisions

## Decision 1 - Official Import Format

The official initial import format will be JSON.

## Decision 2 - Content Creation Workflow

Content may be generated manually or with AI assistance, but all imported content must follow the approved structured format.

## Decision 3 - Import Behavior for Duplicates

The first import tool will skip duplicates and log warnings instead of stopping the entire import.

## Decision 4 - Controlled Growth

The product will grow in phases. The first production target is a strong vocabulary-focused MVP, not a full learning ecosystem.

## Decision 5 - Shared Architecture

The solution must be designed from the beginning for shared business logic across MAUI and future web applications.

## Decision 6 - Localization

The application UI must support localization from the start, even if only a limited number of UI languages are completed in early releases.
